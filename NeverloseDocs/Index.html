NeverloseLibrary

Getting started

Copy this line into your script to load the UI directly:

loadstring(game:HttpGet("https://raw.githubusercontent.com/aoxzdev/NeverloseLibrary/refs/heads/main/Library.lua"))()

or require the module file after downloading it.

Constructor

local MakeUI=require(path.to.NeverloseLibrary)
local ui=MakeUI({cheatname="MyCheat",ext="",gamename="",assetId=12702460854})

Options are all optional.

Tabs & groups

local tab=ui:addTab("Main")
local grp=tab:createGroup("left","Settings")

Control examples

grp:addToggle{flag="opt",text="Option",callback=function(v)print(v)end}

grp:addButton{text="Click",callback=function()print("clicked")end}

grp:addSlider{flag="val",max=100,callback=function(v)print(v)end}

Accessing values

print(ui.flags.opt)
ui.options.val.changeState(50)

Toggle menu

Press RightShift to show or hide the interface.
